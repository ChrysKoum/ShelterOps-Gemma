# 3-Minute Demo Script

## 0:00-0:20 Problem

Emergency shelters receive messy intake notes while volunteers are under pressure. Critical needs, supplies, translations, and handoffs can be missed.

## 0:20-0:40 Solution

ShelterOps Gemma is a local-first operations dashboard powered by Gemma 4. It turns intake notes into triage, action queues, inventory gaps, and privacy-safe public requests.

## 0:40-1:40 Live Demo

Load the flood family demo in `/intake`, analyze with Gemma 4, save the case, open `CASE-014`, show the action queue and translation, then show inventory gaps and generate a volunteer request.

## 1:40-2:15 Architecture

Show `/about-demo`: local Ollama Gemma 4, markdown RAG policies, deterministic urgency scoring, FastAPI, Supabase-ready schema, and in-memory fallback.

## 2:15-2:45 Impact

The system helps small shelters coordinate faster while protecting residents from accidental public disclosure.

## 2:45-3:00 Closing

ShelterOps Gemma keeps the work local, grounded, privacy-safe, and demo-ready even when connectivity or infrastructure is unreliable.
