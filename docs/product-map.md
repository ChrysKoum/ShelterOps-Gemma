# Product Map

ShelterOps Gemma is a local-first emergency shelter coordination dashboard. It converts private intake notes into structured urgency, action queues, supply needs, translations, shift handoffs, inventory gaps, and a privacy-safe public needs board.

The MVP is intentionally not a chatbot. Every screen supports the operations workflow: intake, triage, case work, inventory, needs aggregation, exports, public coordination, and judge-facing architecture explanation.

## MVP Scope

- Private household intake with messy notes.
- Gemma 4 via Ollama analysis with deterministic fallback.
- Deterministic urgency scoring independent of model output.
- Internal cases, case detail, inventory, needs, exports, and settings.
- Public needs board with privacy filtering.
- Demo mode with CASE-014 and Northside School Gym.
